**Koharu-TH** คือสุดยอดโปรแกรมแปลและจัดทำคำแปลมังงะ (Manga Localization Studio) ขับเคลื่อนด้วยระบบ AI ประสิทธิภาพสูง ต่างจากโปรแกรมแปลทั่วไปที่ทำงานเป็นหน้า ๆ ไปอย่างไม่มีความเชื่อมโยงกัน Koharu-TH ได้รับการออกแบบให้ทำงานในลักษณะ **Translation Memory Suite (คลังความจำคำแปล)** ที่สามารถจัดการมังงะทั้งซีรีส์ได้อย่างมืออาชีพ โดยระบบจะจดจำพจนานุกรมศัพท์ (Glossary) ข้อมูลตัวละคร (Characters) และประวัติคำแปลทั้งหมดแบบอัตโนมัติ เพื่อให้คำแปลคงเส้นคงวาและมีโทนเสียงที่ต่อเนื่องตลอดทุกบททุกตอน

แกนหลักของระบบเขียนด้วยภาษา **Rust** ที่ทำงานได้รวดเร็วระดับ Native และใช้ [candle](https://github.com/huggingface/candle) ของ Hugging Face ในการรันโมเดล AI ในเครื่อง (Local ML Inference) พร้อมส่วนติดต่อผู้ใช้ (Desktop GUI) ที่สวยงามสไตล์ Glassmorphism ด้วย [Tauri](https://github.com/tauri-apps/tauri)

## ⚡ 6 พลังหลักของ Koharu-TH (ทำไมต้องเลือกเรา?)

หากคุณกำลังมองหาโปรแกรมระดับมืออาชีพที่มีการจัดการฐานข้อมูลโปรเจกต์เหมือนโปรแกรมแปลระดับสากล (เช่น SDL Trados หรือ OmegaT) แต่สร้างมาเพื่อคนแปลและทำอักษร (Typesetting) มังงะโดยเฉพาะ **Koharu-TH คือคำตอบเดียวของคุณ**

*   🗂️ **Series-Level SQLite Workspace**: บริหารงานแปลมังงะแบบรายโปรเจกต์ (รายเรื่อง) มีระบบเก็บข้อมูลตัวละคร คลังคำศัพท์ และสแกนสกัดรายชื่อศัพท์/ตัวละครจากภาพดิบอัตโนมัติ
*   🧠 **Translation Context (ใหม่ล่าสุด!)**: ระบบดึงบริบทการแปลจากประโยคก่อนหน้าและหน้าก่อนหน้ามาป้อนให้ AI ช่วยแก้ปัญหาคำแปลหลุดโทน หรือตัวละครคุยกันไม่รู้เรื่องเนื่องจากขาดบริบท
*   🤖 **Agentic AI Chat (MCP & Vision)**: ผู้ช่วยอัจฉริยะในแถบด้านข้างที่มองเห็นภาพบนแคนวาสได้จริง มาพร้อมเครื่องมือกว่า 60 ชนิดที่สั่งงานแก้ไขด่านข้อมูลโปรเจกต์ ดึงข้อมูลศัพท์จากลิงก์เว็บวิกิวิชาการ หรือแม้แต่ตรวจไวยากรณ์ไทยสะกดผิดได้ในคลิกเดียว
*   ✍️ **Photoshop-Style Typography**: ปรับแต่งตัวอักษรอย่างละเอียดระดับพิกเซลด้วยระบบจัดระยะบรรทัด (Leading), ระยะช่องไฟ (Tracking), แนวตั้ง/แนวนอน, ตัวหนา/ตัวเอียงแท้ และการหมุนองศาอิสระ พร้อมระบบบีบอัดขนาดอักษรให้อยู่ในฟองคำพูดอัตโนมัติ (Fit to Bubble) และไฟเตือนเมื่อข้อความล้นกรอบคำพูด
*   📦 **Multi-Chapter CBZ Export**: บีบอัดไฟล์แปลเสร็จออกเป็นไฟล์ `.cbz` รายตอนแบบพร้อมอ่าน พร้อมไฟล์แนบข้อมูล `ComicInfo.xml` สำหรับเอาไปเปิดอ่านบนแอปอ่านมังงะทั่วไปได้ทันที
*   🔒 **Enterprise-Grade Infrastructure**: ทำงานราบรื่นเสถียรสูงสุดด้วยระบบฐานข้อมูลสีกรองสำรอง (Fallback DB), การบันทึกไฟล์แบบ Atomic ป้องกันข้อมูลโครงการเสียหาย, ระบบตรวจวัดหน่วยความจำแบบ FFI และการล้างหน่วยความจำคีย์ API เพื่อความปลอดภัยระดับสูงสุด

## ✨ เจาะลึกฟีเจอร์เด่น

### 🗂️ ระบบโปรเจกต์ครอบคลุมทั้งเรื่อง (Series Workspace)
มังงะแต่ละเรื่องจะถูกจัดการเป็นโฟลเดอร์ส่วนตัวที่มีฐานข้อมูล SQLite (`series.db`) ของตัวเอง ซึ่งจะทำหน้าที่ผูกข้อมูลตัวละคร บันทึกราคา และประวัติการแปลทั้งหมดไว้ด้วยกันอย่างเป็นระเบียบ

```
MyMangaProject/
├── series.koharuproj          # ไฟล์ชี้โปรเจกต์ (JSON ขนาดเล็ก)
├── series.db                  # SQLite ฐานข้อมูลหลัก (เก็บตัวละคร, ศัพท์, ความจำคำแปล)
└── chapters/
    ├── ch01/
    │   ├── source/            # รูปภาพมังงะต้นฉบับ
    │   └── render/            # รูปภาพมังงะที่แปลและพิมพ์อักษรเสร็จสิ้น
    └── ch02/...
```

### 🧠 ระบบแปลโดยอ้างอิงบริบท (Translation Context)
บอกลาปัญหา AI แปลประโยคไม่ต่อเนื่อง! เมื่อผู้ใช้สั่งแปลหน้ากระดาษหรือกรอบคำพูด ระบบจะไปดึงคำแปลของประโยคก่อนหน้านี้และหน้าก่อนหน้ามาสรุปแนบไปกับคำสั่งแปลของโมเดลด้วยโดยอัตโนมัติ ทำให้คำแปลสอดรับและลื่นไหลเป็นเนื้อเดียวกันอย่างสมบูรณ์แบบ (รองรับทั้งการแปลผ่านโมเดลในเครื่องและระบบ Cloud)

### 🤖 แชท AI วิเคราะห์ภาพและสั่งการระบบ
แชท AI ด้านข้างไม่ได้มีไว้แค่พิมพ์คุย แต่ทำงานในลักษณะ Agent ที่สามารถเข้าถึงข้อมูลของตัวโปรแกรมได้จริงผ่าน Function Calling:
*   **Vision Support**: สามารถกดดึงรูปภาพของหน้าจอที่กำลังแปลส่งไปให้ AI ช่วยดู และวิเคราะห์ได้ทันที
*   **คำสั่งสแลชหลัก**:
    *   `/fetch-wiki <url>`: ให้ AI ไปดึงข้อมูลสกัดชื่อศัพท์และตัวละครจากเว็บวิกิมาใส่ในฐานข้อมูลให้โดยอัตโนมัติ
    *   `/qc-consistency`: สั่งให้ AI ตรวจสอบความถูกต้องว่าคำแปลในบทปัจจุบันตรงตามคลังศัพท์และชื่อตัวละครที่บันทึกไว้หรือไม่
    *   `/check-thai`: ตรวจสะกด คำทับศัพท์ การวางสระลอย และความสละสลวยของประโยคภาษาไทย พร้อมสั่งเปลี่ยนตามที่ AI แนะนำได้ทันที
    *   `/tm-semantic <text>`: ค้นหาคำแปลเดิมในอดีตด้วยระบบความจำเชิงความหมาย (Semantic Embeddings) แม้คำที่ค้นหาจะไม่ตรงอักษรกันเป๊ะ ๆ

### 🎨 งานศิลป์อักษรที่สมบูรณ์แบบ (Typesetting Tools)
จัดตัวอักษรให้สวยงามประหนึ่งจัดทำใน Photoshop ด้วยเครื่องมือแบบละเอียดยิบ:
*   **การควบคุมขั้นสูง**: ปรับขนาดอักษร, ปรับ Leading (ระยะบรรทัด), Tracking (ช่องไฟ), หมุนองศาสำหรับป้ายเอฟเฟกต์ (SFX) และปรับแนวจัดวางแนวตั้ง/นอน
*   **Fit to Bubble**: ปรับขนาดอักษรอัตโนมัติให้เข้ากับพื้นที่ว่างในกรอบฟองคำพูด พร้อมแสดงป้ายสถานะแจ้งเตือนสีส้ม `TIGHT` (ข้อความแน่นเกินไป) หรือสีแดง `OVERFLOW` (ข้อความล้นกรอบ)
*   **ปักหมุดฟอนต์**: ค้นหาฟอนต์สะดวกรวดเร็วด้วยระบบปักหมุดฟอนต์โปรด (Favorites) ที่จะลอยอยู่ด้านบนสุดของตัวเลือกตลอดเวลา

## 📥 ขั้นตอนการติดตั้งและใช้งาน

Koharu-TH รันบนเครื่องคอมพิวเตอร์ของคุณและใช้กำลังประมวลผลการ์ดจอของคุณเพื่อความเป็นส่วนตัวสูงสุด

### 1. สิ่งที่ต้องติดตั้งล่วงหน้า
*   [Rust Compiler](https://www.rust-lang.org/tools/install) (เวอร์ชัน 1.92 ขึ้นไป)
*   [Bun Runtime](https://bun.sh/) (เวอร์ชัน 1.0 ขึ้นไป)

### 2. ดาวน์โหลดซอร์สโค้ดและติดตั้ง Library
```bash
git clone https://github.com/EarthWL/koharu-th.git
cd koharu-th
bun install
```

### 3. เริ่มรันโปรแกรม
รันระบบเพื่อการพัฒนาและเขียนโค้ดต่อ (รันตัวแอปคู่กับหน้าต่าง UI):
```bash
bun run dev
```
คอมไพล์โปรแกรมออกเป็นตัวโปรแกรมใช้งานจริง (.exe สำหรับ Windows):
```bash
bun run build
```
ตัวโปรแกรมใช้งานจริงจะแสดงอยู่ในโฟลเดอร์ `target/release/`

## ⚡ การ์ดจอที่รองรับและประสิทธิภาพการรัน

ระบบรันโมเดลตรวจจับกรอบคำพูด (Detector), ถอดอักษรภาพ (OCR) และลบอักษรหลังฉาก (Inpaint) บนระบบฮาร์ดแวร์ของคุณ:

### NVIDIA CUDA
*   มาพร้อมกับไฟล์บันทึกโมเดลล่วงหน้าสำหรับ CUDA Toolkit 12.1 และ cuDNN 9.x
*   การ์ดจอที่รองรับ: การ์ดจอ NVIDIA ที่มีค่า Compute Capability **7.5 ขึ้นไป** (สถาปัตยกรรม Turing, Ampere, Ada Lovelace, Blackwell เป็นต้น)
*   *ข้อแนะนำ: อัปเดตไดรเวอร์การ์ดจอของคุณให้เป็นรุ่นล่าสุดเสมอ*

### Apple Silicon Metal
*   รองรับการใช้งานพลังประมวลผลบนการ์ดจอออนบอร์ดของเครื่อง Mac สถาปัตยกรรม Apple Silicon (ตระกูล M1, M2, M3 เป็นต้น) ได้ทันทีโดยไม่ต้องตั้งค่าเพิ่มเติม

### การรันบน CPU
หากคอมพิวเตอร์ของคุณไม่มีการ์ดจอที่ตรงตามเกณฑ์ คุณสามารถสั่งให้แอปประมวลผลผ่านหน่วยประมวลผลหลักได้ผ่านคำสั่ง:
```bash
koharu --cpu
```

## 🛠️ คีย์ลัดและทิปการใช้งาน

*   **เปิดศูนย์คำสั่งด่วน**: กดคีย์ <kbd>Ctrl</kbd> + <kbd>K</kbd> (หรือ <kbd>Cmd</kbd> + <kbd>K</kbd>) เพื่อพิมพ์ข้ามหน้า สลับโปรไฟล์ หรือพิมพ์เรียกคำสั่งแชท AI ได้อย่างรวดเร็ว
*   **การเลื่อนดูแคนวาส**: กด <kbd>Ctrl</kbd> + หมุนวงล้อเมาส์ เพื่อย่อ/ขยายภาพหน้ากระดาษ และกด <kbd>Ctrl</kbd> + คลิกเมาส์ค้างลาก เพื่อเลื่อนดูจุดต่าง ๆ บนหน้าแคนวาส
*   **เพิ่มฟอนต์แปลไทย**: นำไฟล์ฟอนต์สกุล `.ttf` / `.otf` / `.ttc` ไปใส่ไว้ในโฟลเดอร์ฟอนต์ของระบบ (`<app-data>/Koharu/fonts/`) เพื่อให้แอปนำไปใช้งานคู่กับตัวเลือกที่มีอยู่เดิมได้ทันที
*   **เปิดไฟล์ตรงสองคลิก**: บน Windows คุณสามารถดับเบิลคลิกเปิดไฟล์โครงการ `.khr` เพื่อเปิดแอปในโหมดรันหน้าเดี่ยว (Standalone) ได้โดยตรงโดยไม่ต้องกดสร้างโปรเจกต์ใหม่ก่อน

## 🗺️ แผนพัฒนาและสถานะปัจจุบัน

*   **เสร็จสิ้นแล้ว (1.2.x)**: อัปเกรดความเสถียรฐานข้อมูล ป้องกันข้อมูลโครงการทับซ้อน ตรวจเช็คข้อมูลระบบความจำผ่าน FFI, จัดลำดับการอ่านกล่องข้อความอัจฉริยะ (XY-Cut) และเพิ่มสตรีมรูปภาพมังงะเสถียรภาพสูง
*   **มีแผนพัฒนาในรุ่น (1.3.x)**: ผนวกตัวรัน CUDA PTX-JIT เพื่อให้โหลดติดตั้งครั้งเดียวใช้งานได้กับการ์ดจอค่ายเขียวทุกตระกูล, ดึงระบบแปลภาพของ Vulkan มาใช้ และอัปเดตโมเดลลบตัวอักษร AOT Inpaint ตามสถาปัตยกรรมต้นน้ำ
